# 💾 磁盘空间分析工具 v1.0 - 专业版

## 🎯 产品简介

一款专业的磁盘空间分析工具，帮助用户快速识别和定位占用磁盘空间的大文件，支持文件类型过滤、多格式导出等功能，是磁盘清理的得力助手。

## 🚀 快速开始

### 启动程序
**双击 `DiskCleaner_GUI_Stable.bat` 即可启动程序**

### 系统要求
- **操作系统**: Windows 7/8/10/11
- **Python环境**: Python 3.6 或更高版本
- **依赖库**: tkinter (Python自带), openpyxl (Excel导出)

## ✨ 核心功能

### 🔍 智能扫描
- **路径选择**: 支持任意文件夹扫描
- **快速定位**: 一键访问常用目录（下载、桌面、文档等）
- **实时进度**: 独占一行进度条，清晰显示扫描进度
- **扫描设置**: 可设置最小文件大小、最大文件数量、包含隐藏文件

### 📂 文件类型过滤
支持13种文件类型分类：
- **文档类**: PDF, Word, Excel, PowerPoint, TXT
- **媒体类**: 图片 (JPG, PNG, GIF, MP4, MP3)
- **压缩包**: ZIP, RAR, 7Z, TAR
- **程序类**: EXE, DLL, APP, DEB
- **代码类**: Python, Java, C++, JavaScript
- **系统类**: 系统文件、临时文件、缓存文件
- **其他类**: 未分类文件

### 📊 结果展示
- **概览页面**: 显示扫描统计信息和目录分析
- **详细列表**: 按大小排序的最大文件列表
- **多列显示**: 文件名、大小、路径等详细信息

### 📄 多格式导出
- **Excel格式**: .xlsx文件，自动用Excel打开
- **CSV格式**: .csv文件，Excel兼容，自动打开
- **HTML格式**: .html文件，浏览器打开，支持交互式排序

### 🎨 界面设计
- **专业布局**: 功能分区清晰，操作简单直观
- **居中按钮**: 功能按钮居中显示，界面美观
- **响应式设计**: 自适应窗口大小
- **中文友好**: 完美支持中文显示和输入

## 📖 使用指南

### 基本使用
1. **启动程序**: 双击 `DiskCleaner_GUI_Stable.bat`
2. **选择路径**: 手动输入或浏览选择要扫描的文件夹
3. **快速选择**: 使用快速访问按钮直接选择常用目录
4. **设置条件**: 调整最小文件大小、最大文件数等扫描参数
5. **文件过滤**: 选择要扫描的文件类型（可选）
6. **开始扫描**: 点击"开始扫描"按钮
7. **查看结果**: 在"概览"和"最大文件"标签页查看结果
8. **导出数据**: 点击"导出结果"选择导出格式

### 高级功能
- **文件类型过滤**: 勾选特定文件类型进行精准扫描
- **批量操作**: 支持全选/清除文件类型过滤器
- **隐藏文件扫描**: 可选择包含系统隐藏文件
- **智能暂停**: 支持随时停止正在进行的扫描

## 🛠️ 技术特性

- **多线程处理**: 扫描过程不阻塞界面操作
- **异常处理**: 完善的错误处理机制
